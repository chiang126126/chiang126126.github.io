# BTC_Alert_ · 跨月总结（2026-01 ~ 04）

## 人物定位

BTC_Alert_ 是典型的**宏观 + 跨资产配置型**交易者，与纯链上叙事型 KOL 有本质区别。核心特征：
- **多资产并行**：BTC、黄金、稳定币（USD1）、perp DEX、盘前套利同时运行
- **自建量化体系**：有正在迭代的自动化工具（基差 + OI + 现货集中度）
- **守正风格**：拒绝追热点流量，优先系统性一致性（「求新的很多，守正的不多也不容易」）
- **经历驱动迭代**：edgeX 老鼠仓 → 对冲体系必要性验证；$genius 爆仓 → 纪律 vs 体系反思

## 核心交易框架提取

### 框架一：跨域资产配置体系（宏观层）

```
BTC（高风险，卖在高位 12.4w）
    ↓ 转仓
黄金（避险层，成本 3,100，长持）
    └── 逻辑：美元储备货币地位崩塌 + 无序交易溢价
USD1（稳定收益层，超国债 APR）
    └── 逻辑：实物美元抵押 + 政治背书 + 平台补贴（WLFI）
```

**配置转换触发条件**：
- BTC → 黄金：市场开始计价美元信用崩塌，地缘风险上升
- 黄金 → 不动：长持逻辑未变（无序+通胀+地缘）则不止损不追高
- 黄金部分止盈条件：FOMO 情绪过重（非逻辑改变）
- USD1 出场条件：WLFI「中选事件」落地后的抛压一次性释放时减仓

**黄金冷战类比框架**（03月）：
```
科技发展停滞 → 地缘对抗 → 资源争夺（稀土/钨）→ 战略溢价
→ 最终握手言和 → 科技驱动重组 → 世界金融秩序重置
当前位置：威胁阶段（比预期更激进），黄金仍在定价混乱溢价
```

### 框架二：盘前套利层级体系（执行层）

**平台选择矩阵**：

| 平台 | 适用场景 | 规模上限 | 风险 |
|------|----------|----------|------|
| @aspecta_ai | **有币方锁利润** TGE 前/等交割 | M 级别 | 无爆仓风险 |
| PM | 事件性市值预测 | 小~中 | 需等揭晓才兑现 |
| WhaleMarket | 小仓位试水 | 几万刀 | 深度不足大单 |
| CEX 盘前 | 不适合有币方 | 无限制 | 波动极大，经常给项目方送钱 |

**盘前锁利润操作流程**：
1. 确认有空投（已持仓/参与撸毛）
2. 评估 TGE 前价差：asp 价 vs 盘前预期
3. 在 asp 建空单（等 TGE 交割）或 asp 空 + CEX 多对冲
4. **关键原则**：「多大尺寸带多大的套，不然容易滑套」—— 对冲仓位 ≤ 已持有空投价值

**价差套利公式**（$chip 案例）：
```
asp 多（0.02911）+ CEX 空（0.03726）→ 价差抹平后平 CEX 空
→ 持 asp 等交割（0.057）→ 约 1 倍收益
```

| 时间 | 开仓价 | 平仓价/逻辑 |
|------|--------|-------------|

**出仓决策框架**：
- 长线空单：「预期通过震荡消化 → 继续持有浪费时间 → 平仓」
- 不平仓条件：「一以贯之的交易体系支撑」（非他人意见，需有明确依据）
- 止损思路：出游等特殊情况 → 挂止损保护，不强行盯盘

### 框架四：自动化交易工具体系（量化层）

**信号条件（迭代过程）**：

```
v1（基础版）：
  1. 基差变化速率（基差扩大 = 主力建多仓）
  2. OI 积累判定（单边 vs 双持）

v2（优化版）：
  + 现货集中度（>60% = 出货信号或控盘信号）
  → 胜率 72-84%

v3（当前）：
  + 交易所限价风控预警（妖币保护）
  → 胜率 84-86%
```

**进场信号**：
- 基差**扩大** + OI **单边积累** + 现货集中度**低** → 主力建多 → 做多
- 基差**收窄** + 现货集中度**突然升高** → 主力出货 → 做空 or 出场

**出场信号**：
- 现货集中度 > 60% + 资金费率过热（> 0.08%）→ 立即止盈（多单）
- 交易所风控介入（如 OKX 限价）→ 流动性即将枯竭 → 立即平仓（空单）

### 框架五：「狂战士后入法」纪律框架（心态层）

从狂战士 50 万盈利案例中提炼的可复制框架：

**后入两大原则**：
1. **看不懂的不后入**：不理解逻辑的标的，哪怕价格在涨也不进
2. **群友垫的不够多的不后入**：等先行者承担初期风险后，用已验证的信息进入

**适用场景**：
- 行情已经启动但未到高潮
- 先入者（群友/博主）已经公开了具体操作和逻辑
- 自己可以用后入价位设置清晰止损

**核心启示**：复杂体系 + 差情绪 < 简单原则 + 极强纪律。在群体行为可观察的市场中，后入往往比先入更高胜率。

### 框架六：第一性原则选币框架（01月，研究层）

适用于选择新赛道早期标的（如 $ROLL Base perp DEX）：

```
1. 赛道是否真实需求且当前空白？

4. 交易量走势是否支撑上所预期？
满足以上 → 持有等待上所催化
```

## 可用于 DeepSeek 系统提示的判断规则

### 盘前套利入场规则

```
盘前套利机会判断（满足以下全部）:
1. asp 盘前价 vs CEX 开盘价 有明确价差（>15%）
2. 有已持仓代币（空投/参与者），即「有币做套利」
3. 对冲仓位不超过已持仓价值
4. 项目有 TGE 交割机制（不是无限期盘前）

执行：asp 多 + CEX 空 → 价差抹平后平 CEX 空 → 等 TGE 交割
```

### 自动化工具做多信号

```
做多信号（需同时满足）:
1. 基差扩大速率加快（现货指数 < 合约价差扩大）
2. OI 积累为单边建多（而非双持）
3. 现货集中度 < 40%（无出货迹象）
4. 资金费率处于正常范围（< 0.05%，未过热）

出场信号（任一满足即出）:
1. 现货集中度 > 60%
2. 资金费率 > 0.08%
3. 交易所触发限价风控
```

### 黄金/宏观仓位调整触发器

```
继续持有黄金（长线逻辑检查）:
✓ 美元储备货币地位压力仍在
✓ 地缘未完全缓和（伊朗/霍尔木兹仍有博弈）
✓ 通胀/债务逻辑未逆转

减仓黄金信号（任一满足）:
- FOMO 情绪极度过热（如年内多个博主同时喊牛）
- 地缘显著缓和（如真实停火协议签署）
- 美元指数显著走强超预期

不抄底 BTC 的持续条件（未触发以下全部则不买）:
- 没有大资金入场信号（ETF 净流入转正且持续）
- 没有出现更大跌幅（定义：从高点回撤 > 30%）
- 没有恐慌性清算（链上爆仓 > 10亿 + 情绪极度悲观）
```
