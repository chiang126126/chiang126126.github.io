# EmberCN · 跨月总结（2026-01 ~ 04）

## 人物定位

EmberCN 是 **P2 扫描模型的核心信号来源**，纯链上数据陈述型分析师：
- **不做方向预测**：只陈述「X 地址做了 Y 操作」的客观事实
- **持续追踪大户**：Arkham / DeBank / Hypurrscan 是主要工具
- **模式识别**：通过重复追踪，形成对项目方/巨鲸/财库公司的「行为画像」
- **最大价值**：提供决策所需的「事实锚」，是所有主观判断的最可靠信号源

## 核心链上行为分类体系

### 类别一：财库公司（长期机构买入）

```
跟踪对象:
- Strategy (MSTR): BTC 财库，持仓 81.5 万枚，成本 $75,527
- Bitmine (BMNR): ETH 财库，持仓 497 万枚，成本 $3,596
  特征: 70% 强制质押
- FG Nexus: ETH 财库（已亏损 $8,700 万）
- Sky 财库 (SDEV): SKY 持仓 22 亿枚

信号解读:
- 每周持续买入 → 长期看多（尤其 Strategy）
- 单周破纪录买入 → 阶段性底部信号（如 Strategy 4月 34,164 BTC）
- 70% 质押标准 → Bitmine 建立了机构持仓范式
- 财库公司浮亏扛仓 → 成本均价附近的支撑参考
```

### 类别二：远古巨鲸（长期抛压）

```
典型案例:

  到 4月累计卖出 4,000 枚，实现收益 $3.65 亿
- 2014 BTC 远古（8,000 枚）：3月完成清仓
  累计收益 $6.8 亿
- 2015 ETH ICO 巨鲸（@$0.311）：
  部分已清仓获利 $2.69 亿
  仍有 17-20 万枚 ETH 活跃

信号解读:
- 远古加速卖 → 顶部信号
- 远古持仓换仓（BTC→ETH）→ 宏观偏多 ETH
- 远古波段操作 → 震荡市常态
```

### 类别四：项目方抛售（链上卖压）

```
典型抛售路径:
1. 多签钱包 → CEX 热钱包 → 分批卖出
2. 跨链（如 BLESS: Sol → BSC）抛售
3. OTC 通过 Wintermute/FalconX 场外处理
4. BitGo 托管钱包拉升后转出

本轮典型案例:
- TRUMP 团队: 2月累计抛售 $5,700 万 → -40%
- WLD 项目方: 历史新低还继续 OTC 卖 → 底未到
- Bless: 3亿枚跨链抛售 → -71%
- AKE: 55% 流通量转 Binance Alpha → -65%
- RAVE: 2300 万枚转 Bitget → -46%(4h)
- Aztec 项目方: 1.9万 ETH 公募所得全部变现
- Ondo/Worldcoin: 多签持续转 CEX

信号强度:
- 新低价位卖 > 下跌趋势卖 > 反弹卖
- 跨链抛 > 直接卖（规避监控）
- 凌晨拉升后立即转 CEX = 典型诱多出货
```

## 可用于 DeepSeek 系统提示的判断规则

### 做多信号判断（多信号组合）

```
强做多信号（满足 3 项以上）:
1. 财库公司（Strategy/Bitmine）单周创纪录买入
2. 项目方启动回购计划（Lido/Spark 案例）
3. 大鲸鱼从 Aave 等迁移至积极协议（Spark）
4. 大额杠杆空头止损平仓（short covering）
5. 平台 SAFU 级别的大规模买入（Binance $10.5 亿）
6. 远古巨鲸停止抛售

弱做多信号:
- 巨鲸从 BTC 换成 ETH（相对补涨逻辑）
- Bitmine 70% 质押买入（锁定长期抛压）
- OTC 跟随项目方回购买入
```

### 做空/出场信号判断

```
强做空信号（满足 2 项以上）:
1. 项目方在历史新低位继续 OTC 卖出（WLD 模式）
2. 远古巨鲸加速抛售 + 大资金杠杆多头清算
3. 控盘 > 60% 的币出现深夜拉升 + 托管钱包转出（诱多出货）
4. 多个项目方钱包同时多签转 CEX
5. 季度解锁卖压（Chainlink 案例）

出场信号（已持仓）:
- 项目方从多签转 CEX 开始
- 控盘度 > 93% 后开始拉盘（SIREN 最后阶段）
- 主动做市商暴涨后减仓
- 大额鲸鱼止盈平仓
```

### 项目方行为 - 控盘度风险阈值

```
控盘度 < 30%: 相对安全（流动性合理）
控盘度 30-60%: 中等风险（需关注大户动向）
控盘度 60-93%: 庄家游戏场（严格避免长线）
控盘度 > 93%: 纯操盘（只做极短线 + 提前撤退）

识别工具:
- 单地址持币比例（Arkham Intelligence）
- 代币 Top 100 持仓集中度
- 归集行为（短期内多钱包汇集）
```
