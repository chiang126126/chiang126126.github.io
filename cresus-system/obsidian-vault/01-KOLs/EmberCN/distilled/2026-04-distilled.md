```

```

Step 4: 小额通过 Umbra 隐私协议转移

THORChain 受益：
- 24h 交易量 $2,000 万 → $3.6 亿（18 倍）
- 24h 手续费 $5,000 → $42 万（84 倍）
- 总计为黑客处理了 75,700 枚 ETH
- THORChain 收入 $91 万
```

```
