## 核心交易框架

### 框架一：「垃圾时间 vs 蜜糖时间」市场状态识别

```
垃圾时间特征：
- 价格经历大幅拉升后无力继续
- 横盘震荡，假突破假跌破频繁
- 交易结果：耗精力，无胜算（与野猪摔跤）

蜜糖时间特征：
- 品种有清晰方向性和动力
- OI + 成交量同步放大
- 趋势线被有效突破

操作规则：
- 垃圾时间：轻仓/不操作，等待
- 蜜糖时间：重仓跟随
```

---

### 框架二：期权策略全周期应用

**熊市底部阶段（当前）**：
```
1. 阶梯建仓现货（65K→60K→55K→50K→45K→40K，每跌5000建一单）
2. Sell Put（30-50%保证金，不同价位阶梯卖，接货）
   - 横盘收敛 + IV 走低时暂停
   - 砸盘后恢复卖
3. 目标：低成本积累 BTC 现货

牛市初期（2026 Q4）：
4. 买入 LEAPS Call（年化成本 10-20%）
   - 到期 1-3 年最长期
   - 工具：Greeks.live 查年化成本
   - 深度实值 或 虚值期权均可（年化成本 10% 左右）

牛市顶部区间：
5. Sell Call（备兑减仓）
   - 阶梯价格分批操作
   - 对应现货逐步减仓

止盈节点：
6. 2029 年 Q3 基于时间分批定量减仓
```

**策略优势**：
- 现货越来越少 → LEAPS 越接近实值 → 实质杠杆多头而风险递减
- 尾部风险 LEAPS(最大 10-20%/年) << 合约（可爆仓）
- 7x24h「天地单」心理按摩 + LEAPS 安稳睡眠

### 框架三：信号系统（OI + 成交量 + 清算水平）

**核心信号**：OI 与成交量同步放大 = 真实主力入场

**工具**：Leviathan「Liquidation Levels」TradingView 指标
- 检测大额新仓位涌入 → 估算强平价格 → 画出清算水平（黄点）
- **黄点 = 高确信度支撑/压力位**

**操作**：
```
到达黄点时：
  - 买入价格反转期权（低风险，有限亏损）
  - 或开合约单：止损 = ATR 指标，止盈 = 黄点
```

**辅助判断**：
- 价格涨 + OI 降 = 空头被爆仓后无新空头 → 主力了结多单 → 做空信号
- 费率转绿 + OI 降 = 空头撤退 → 做多信号（short covering 推涨）

### 框架四：MSTR mNAV 终极抄底法则

```
MSTR mNAV = MSTR 市值 / MSTR 持有 BTC 总价值

mNAV < 1.0 → 打折买 BTC（附赠软件公司）→ 历史最优抄底时机

应用：
- 不看 BTC 价格，看 MSTR 折价程度
- mNAV 折价越深 = BTC 底部信号越强
- 历史案例：60K 时 mNAV < 0.9，随后反弹 30%+

BTC 底部判断辅助：
- 平底场景：震荡在 60K（横盘磨底）
- 尖底场景：快速下杀至 40K（为下轮牛市夯基础）
```

### 框架五：加密概念美股配置（山寨替代）

## 可用于 DeepSeek 系统提示的判断规则

### BTC 市场状态判断

```
垃圾时间信号（任意两项满足→轻仓等待）:
1. 价格经历大幅拉升后无力继续上涨
2. OI 和成交量未同步放大
3. 假突破假跌破频繁（趋势线失效）
4. 稳定币市占率上升（Ben Cowen 指标）

蜜糖时间信号（任意两项满足→可加仓）:
1. OI + 成交量同步放大
2. 日线下降趋势线被有效打破
3. MSTR mNAV 折价 < 0.9
4. 空头撤退（费率转绿 + OI 下降 + short covering）
```

### 期权做空信号（山寨币层面）

```
做空条件（满足以下全部）:
1. 价格持续上涨
2. OI 同步下降（空头被爆仓，无新空头接力）
3. 可能出现负费率（准备吸引多头接盘）

出场条件:
- 等待新一轮负费率吸引多头进场后，在主力建空完成后空仓
- 止损：前高上方
```

### BTC 抄底终极指标

```
终极信号: MSTR mNAV < 1.0
  → 买入时机，不看 BTC 价格

辅助确认:
- CME 期货缺口已填补
- 日线下降趋势线触碰并反转
- 熊市回吐 > 60% 牛市涨幅
```
