### 决策 1：「网格天地单 + LEAPS Call」复合策略（核心框架）

这是 CryptoRounder 本月最重要的完整策略，横跨多年时间维度。

#### 第一层：网格天地单（现货层）

- **本质**：超长期做多策略，随价格上涨持续减仓
- **优化点**：
  1. **入场时机决定成败**：必须在低位开启，越高开效果越差
  2. **资金利用率问题**：越接近「天」，现货越少，最终跑输单纯持仓
  3. **核心价值**：心理按摩——7x24h 不断套利产生「一直在盈利」的感觉，让人熊市扛住不割
- **建仓参数**：
  - 阶梯建仓：从 65,000 开始，每跌 5,000 建一单，最低建到 40,000
  - 网格范围：30,000–200,000
  - 网格数量：所在平台最大值

#### 第二层：长期看涨期权 LEAPS（杠杆层）

- **核心工具**：深度实值 Call，到期 1–3 年
- **年化成本目标**：10–20%（低于牛市合约做多的资金费率）
- **买入时机**：
  - **2026 年第四季度**分批买入最长期期权
  - 基于 BTC 4 年周期规律（「每个周期都说这次不一样，但总是一样」）
  - 过早买入会持续损失时间价值，Q4 是最优时间窗口
- **移仓**：加密货币平台通常无 3 年期期权，到期前 1–2 个月换仓更远期
- **验证工具**：Greeks.live → 查看「年化成本」估算最大年损失

#### 整体止盈时机

- **2029 年第三季度**，基于时间分批定量减仓

#### 策略优势

- 现货越来越少 → LEAPS 越接近实值 → 实质上持有杠杆多头而风险越来越低
- 尾部风险显著低于合约：LEAPS 最大损失 10–20%/年，合约极端情况可爆仓
- 享受 3 年「天地单」震荡收益 + LEAPS 安稳睡眠

### 决策 2：Sell Put 熊市生息策略

- **适用行情**：震荡下跌行情（比裸空更舒适）
- **执行逻辑**：
  1. 当前熊市接近底部区间 → 用 30–50% 资金挂单建仓现货
  2. 用另外 30–50% 保证金在不同价格卖 put 接货（阶梯卖法）
  3. 横盘收敛、IV 走低时暂停 → 砸盘后再继续卖
  4. 未来牛市高位反过来 sell call（备兑减仓）
- **净值曲线效果**：平滑，避免主观赌方向的大幅波动
- **Sell Put vs 裸空**：震荡下跌时裸空难持有（假突破太多），sell put 更舒适
- **回测数据依赖**：@Blank_TX 开源数据，增强对「持续流血」焦虑期的信心
