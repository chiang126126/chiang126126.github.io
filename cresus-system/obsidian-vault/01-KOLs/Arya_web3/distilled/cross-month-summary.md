# Arya_web3 · 跨月规律总结（2026 Q1 全量）

> 基于 2026-01 ~ 2026-04 四个月蒸馏笔记提炼的稳定规律、核心框架和操作系统。

---

## 一、Arya 的核心认知体系（贯穿全年）

### 1. 市场本质观：赌场 + 庄家博弈

- 币圈本质是赌场：做市商（MM）、项目方、交易所三方协同，散户是燃料
- **不是预测涨跌，而是预测庄的意图**
- 不同阶段只有两类机会：**Alpha 上新** 和 **庄币跟庄**；现货抄底大概率妙瓦底（4月明确表态）
- 币安自身可能就是最大的庄（#币安人生 案例）

### 2. 赚钱的第一性原理

> "财富的第一性原理是积累，而不是赌博"（1月）

- 每次交易赚 1-2x，不断重复，比追高倍数更可持续
- 落袋为安 > 贪心（庄币七原则核心）
- 在自己舒适区赚钱，不断重复

## 二、稳定出现的决策模式（≥2月重复）

### 模式 A：币安 Alpha 三段锦出货法（1月、3月均现）

完整框架见 [2026-01-distilled.md](2026-01-distilled.md)。

**核心路径**：Alpha 低开买入 → 等合约 → 分三段出货（达到 1-2x/合约上线 FOMO/韩国所预期破灭）

**关键指标**：
- 项目方是否有主动做市商（主动 MM 存在是必要条件）
- 合约上线速度（day1 vs day7+ 代表项目方实力）
- Aster DEX 是否早于币安上合约（时间差套利窗口）

---

### 模式 B：MM 爆空识别与跟单（1月雏形 → 4月成熟体系）

1月已提出主动/被动 MM 识别；4月针对 $Rave/$Siren/#币安人生 做了最完整的机制拆解。

**核心指标组合（每次必看）：**

```
OI 变化方向 × 多空比 × 爆仓数据规模 × 资金费率节奏
```

| 场景 | 信号组合 | 操作 |
|---|---|---|
| MM 爆空启动 | OI 增加 + 空军 > 多军 2-3倍 + 资金费率为正 | 跟多 |
| MM 高位运行 | 持续爆仓小额空单 + 大额空单未出 | 持有，等大额空单出现后准备出 |
| MM 收网 | OI 下降 + 多单撤 + 资金费率变负 + 现货砸盘 | 清多 / 开空 |
| 数据可疑 | BG/BingX 占比大 + 无 100k+ 单子 | 观望，刷量数据不可信 |

---

### 模式 C：Coinbase 路线图 → 币安合约路径（1月 $elsa、3月 $PRL 均验证）

**触发条件**：被 Coinbase 列入上币路线图
**完整路径**：Alpha 低开 → Coinbase 现货 → 币安合约 → 韩国所
**操作时机**：Alpha 首发时低位建仓，合约上线 FOMO 时出部分（或在 Aster 提前埋伏）

---

### 模式 D：赛道龙头崛起后抢龙二 TGE（3月提出，反面教材来自 perp DEX）

**规律**：赛道龙头大涨（如 $Aster 被 CZ 喊单）后，同赛道尚未 TGE 的标的是最佳布局窗口
**操作**：龙头起来后立刻扫描同赛道 TGE 候选，龙二先 TGE 者优先入场
**风险**：项目方不懂择时，拖延 TGE 错过窗口（$edge $bp 的悲剧）

## 四、数据维度依赖图谱

Arya 全季度最依赖的数据源（从高到低）：

1. **OI（持仓量）及其变化方向** — 每篇必看，是最核心的庄家意图指标
2. **多空比（账户数多空比 vs 名义多空比）** — 判断爆空性价比
3. **爆仓数据（金额 + 单笔大小 + 交易所分布）** — 判断 MM 真实操盘 vs 刷量
4. **资金费率（尤其是 1小时费率节奏）** — 判断变盘临近时机
5. **链上地址行为（庄家接货地址、项目方地址）** — 进场时机
6. **交易所上币节奏（Alpha → 合约 → 韩国所的天数）** — 出货时机
7. **Coinbase 路线图公告** — 优质 TGE 候选筛选

## 六、可直接输入 DeepSeek 的判断框架（P3 AI 层参考）

以下是从 Arya 4 个月帖子中提炼的、可量化的判断规则，供 DeepSeek system prompt 使用：

```
【Arya_web3 核心框架 - 做市商操盘识别】

买入信号（满足 3 项以上）：
1. OI 在过去 24h 增加 > 50%，且多单主导
2. 多空比（账户数）< 50%，空军是多军 2 倍以上
3. 24h 爆仓以空单为主，且有 > 10 万美元的单笔空单
4. 资金费率为正且稳定（非异常高）
5. Binance 是 OI 主力交易所（> 40% 占比）
6. 链上：庄家地址净接货，项目方地址无出货

卖出信号（满足 2 项以上）：
1. OI 在过去 6h 下降 > 20%，多单明显减少
2. 资金费率由正转负
3. 大额空单未出现，MM 收益不够（拉盘会暂停）
4. 现货在 gate/BG 有地址砸盘
5. 爆仓单集中在多单（MM 已完成空单爆破转向爆多）

排除标的：
- vol/OI > 20x（OI 太小）
- 三无项目（无融资/无背书/无做市商）
- perp DEX 赛道（2026 Q1 以后基本死亡）
- 被 Binance 合约下架的标的
```

## 七、Arya_web3 的局限性（蒸馏者注）

- **时效性强**：帖子以短线操作为主，框架依赖当时行情，不适合做多周期验证
- **幸存者偏差**：只分享成功案例和复盘失败，但已大幅避开追涨杀跌
- **信息优势**：部分机会来自私下人脉（群友投喂），纯数据层面难以完全复制
- **风格标签**：适合做市商主导的短线庄币和 Alpha 机会，不适合长期价值投资
