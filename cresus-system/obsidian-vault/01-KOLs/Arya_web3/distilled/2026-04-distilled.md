## 交易决策提取

### 决策 1：MM 爆空全流程识别与跟单框架

> 本月核心，Arya 对 MM 爆空机制做了迄今最深入的系统性拆解（$Rave、$Siren、#币安人生 三个案例）。

**MM 爆空的完整运作机制：**

1. **积累空单阶段**：拉盘进涨幅榜 → 吸引注意力 → 横盘制造要跌假象 → 空军陆续进场
2. **定点爆破阶段**：识别到足够大的空单后 → 直接拉盘爆掉，一笔接一笔（8u→9u→10u）
3. **诱空循环**：爆完一波空单 → 横盘或小回调 → 空军以为要收网再进场 → 再次拉爆
4. **收网信号**：资金费率变负且空单已扛不住 → MM 可能一键 sell 变盘；或多单大幅减少、OI 下降、现货砸盘

**关键判断指标（按重要性排序）：**

| 指标 | 判断逻辑 |
|---|---|
| OI 变化 | OI 翻倍 + 多单增加 = MM 开启爆空；OI 下降 + 多单减少 = MM 开始收网 |
| 多空比 | 空军是多军 3 倍以上（如 36.5%、27%）= 爆空性价比极高 |
| 爆仓数据 | 大额单子（100k+）= MM 真实操盘；全为小额（10k-30k）= 刷量居多 |
| 资金费率 | 1小时收取且费率高 = 空军扛不住变盘临近；变负 = 危险信号 |
| 全网 OI 分布 | BG/BingX 占比大 = 大概率刷量；OKX/Binance 为主 = 真实交易 |

**跟单原则：**
- 确认 MM 主力在多头方向后，跟多不跟空
- 入场价格越接近 MM 平均多单成本越安全（如 #币安人生 MM成本 0.16u，入场需接近此价）
- 见到知名 KOL 喊做空 = MM 最喜欢的空单燃料，反向参考

- **涉及帖子**：[04-11~04-16 多篇](https://x.com/Arya_web3/status/2043549689036116213)、[04-13](https://x.com/Arya_web3/status/2043708001664880675)

### 决策 2：MM 收网信号识别（做空时机）

> 与决策 1 互补，识别 MM 从多头转向收网的信号，可在此时开空。

- **触发信号（满足 2-3 个即可考虑开空）**：
  1. 多单 OI 在减少，空单没有变化，整体 OI 下降
  2. 资金费率由正变负或持续扩大
  3. 该标的现货在 gate/BG（庄惯常挂牌所）有地址转入并砸盘
  4. 拉盘后无大爆仓单出现（说明空军不多，MM 收益不足，需回调再诱空）
- **注意**：并非所有条件同时满足，抓住 OI + 多空比 + 资金费率核心三项
- **涉及帖子**：[2026-04-11](https://x.com/Arya_web3/status/2043003456890507587)

### 决策 3：庄币操作七原则

> 本月归纳的完整庄币操作框架，适用于所有 MM 操盘标的。

1. **做短线**，不做长线
2. **见好就收**，不追高倍数，不从头吃到尾（基本不可能）
3. MM 的对手盘和跟车者都是 MM 目标，有各种方式洗你和爆仓你
4. **不过夜**：不盯盘就不做，或设好止损止盈，否则空仓
5. **有耐心**：等到确定性点位，没有就休息
6. 结合**链上/合约数据 + K线 + MM 手法**，总结出该 MM 的操盘风格和诱空手法
7. **落袋为安**，贪婪了你就是鱼

> 复盘 $opg $chip 失误，提炼高流动性市场下的 Alpha 操作调整。
