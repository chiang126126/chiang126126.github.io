### 框架七：市场结构与流动性

```
关键判断：
  流动性 > 生产力（本轮周期底层逻辑）
  Token 需要「央行」角色（流动性托举 + 叙事锚定）
  没有央行角色 → 社区自发 → 散户互割

Memecoin 长命必要条件:
- 有 CEX 托举（盘面不够大）
- 或 Ponzi 类大池子（套保需求强）

社区/地推机制:
- 团队长入金质押 → 不敢随便卖（社区监控）
- 社区长倾向扩大市场（离核心越远越能套现）
```

## 可用于 DeepSeek 系统提示的判断规则

### 妖币做空信号（V4A-Flash 简化版）

```
入场条件（全部满足）:
1. 72H 内有超出大盘普涨的单独拉盘
2. 4H 线出现明显上影线（抛压确认）
3. 1H 收盘价从上影线前高回撤确定百分比
4. 跨所 OI 集中度：Binance 占比 < 中位数
5. vol/OI < 20x（避免纯刷量诱多）

加强信号（任意 2 项）:
- 链上出现项目方/大户流入交易所
- 30min 内 OI 骤然减少（插针爆仓信号）
- 连续数小时价涨 OI 跌（V8 背离）

出场策略:
- 猜错方向：1H 收盘价 10% 止损
- 猜对后反转：紧 trail 平仓
- Trail 出场 8H 后继续跌 → 再入场（更低止损）
```

### 宏观配置规则

```
硬资产做多配置（核心）:
- 黄金：M2 增速 > 生产力增速时配置
- BTC：战争/地缘风险上升时加仓
- 稀缺矿产 / 能源：法币重构期价值兑现

不做空硬资产信号:
- M2/Gold 汇率下破新低（美元持续贬值）
- 债务 / GDP 比例 > 3.0（无法抵消通胀）
- 重大地缘冲突（避险需求硬通货）

减仓硬资产信号:
- AI 生产力指标突破 M2 增速
- 美联储明确转鹰 + 实质性缩表
- 地缘风险显著缓解
```

### 平台选择规则

```
避开:
- Hyperliquid 合约（MM 竞技场，散户无胜算）
- 资金费率连续 3 天 > +0.08% 的币（MM 诱多）

优先:
- BSC / Binance 生态（东方叙事根基）
- 有「央行角色」托举流动性的项目
- 跨所 OI 分散的妖币（可操纵）做空

新闻交易纪律:
- 信号源 → 秒级验证
- 预设一键买入（不临时调参数）
- 阶梯分批卖出（33% / 10%）
- 接受小亏立刻下一个
```
