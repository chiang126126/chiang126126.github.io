### 决策 1：Hertzflow 信号系统性复盘（核心发现）

这是 thecryptoskanda 本月最重要的**量化发现**，具备普适价值：

#### 币圈合约端发现
```
1. 做空胜率 ≈ 做多胜率 × 2（两倍）
2. 高流动性交易对（币安合约成交量 Top 50）胜率比小对高 10%
3. 绝大多数异动信号出现在北京时间凌晨 5:00-7:50
   = 美盘收盘 + 欧洲收盘 + 亚盘开盘前
   = 全球流动性最低时段
```

#### 大宗商品端发现
```
绝大多数信号出现在北京时间 7-8 点
= CME 现金盘收盘 + 夜盘开盘
= 同样流动性最低时段
```

#### 核心结论
**信号出现在流动性最低时段** = MM/庄家选择低流动性时操作（更容易达到目标）

#### 操作启示
- 优先做空而非做多
- 只做高流动性交易对
- 重点监控流动性最低时段的异动
