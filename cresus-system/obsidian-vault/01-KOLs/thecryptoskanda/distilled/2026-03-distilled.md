## 交易决策提取

### 决策 1：Memecoin 地推机制（社区锁定 vs 抛售）

这是对 memecoin「为什么建设不起来」的核心洞察：

**案例**：项目方团队长讨论
- 团队长质押入金后**不敢随便卖**
- 原因：下面市场都盯着这个地址，出问题要负责
- 地推不像网推，邀约客户都是人带人，出问题解决也是人找人

**社区长的动机**：
```
把市场做得越大、越扩散、越不相关、甚至做到海外
↓
离核心团队越远
↓
他们越能套现、风险越小
```

**意义**：理解 memecoin 社区背后的真实动机 → 越早期越集中，越后期越分散

- **涉及帖子**：[2026-03-30](https://x.com/thecryptoskanda)

### 决策 3：交易对流动性判断（多空对手盘）

- **核心原理**：任何交易对，多空流动性没了就死盘
- **细分分析**：
  ```
  Memecoin：
  - 盘面太小 + 参与者少
  - 没有 CEX 托举基本不长命

  Ponzi（大池子）：
  - 池子越大 → 套保需求越强
  - 对手盘流动性可持续
  ```
- **操作启示**：做多 Meme → 必须有 CEX；做 Ponzi 类 → 看池子规模
- **涉及帖子**：[2026-03-29](https://x.com/thecryptoskanda)

### 决策 4：澳股 ASX 作为炒币延伸

- **推荐**：炒币能力强的可以看看澳股 ASX
- **三大理由**：
  1. 世界少有的以**资源股**为龙头的发达国家股市
  2. 澳洲是全球最大的铁矿石、LNG、铀矿、铜矿、金矿、稀土产地
