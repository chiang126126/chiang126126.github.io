## 交易决策提取

### 决策 1：强控盘币中后期全面回避

- **核心判断**：$SIREN、$RAVE 等强控盘币，一旦全网注意力集中其上，**所有技术指标失效**
  - 资金费率失效
  - 基差失效
  - OI 价格背离失效
  - 原因：庄家主动制造这些信号来诱导散户
- **结论**：强控盘中后期 → 不进入 → 保本金
- **$SIREN 操作复盘**：
  - 横盘吸筹 2 周后入场（吃稳健段位）
  - 1.5 先跑（风险信号出现即离场），未吃到最高位
  - 「钱是赚不完的，赚自己应该赚的就好」
- **判断依据**：蓄水池 + OI 结构，若「油已被榨干」则不进
- **涉及帖子**：[2026-04-16](https://x.com/derrrrrrrq)、[2026-04-17](https://x.com/derrrrrrrq)

### 决策 2：CVD 信号局限性（重要认知修正）

- **传统认知（错误）**：CVD 上升 = 买盘强 = 看多
- **derrrrrrrq 修正**：
  1. CVD 上升 + OI 上升 + 价格上升 = **可能是庄家建空头仓位**（CVD 是烟雾弹）
  2. CVD 减少不代表看空，CVD 增加不代表看多
  3. CVD 仅在**清算时刻**有参考意义
- **正确用法**：
  - 关注 CVD 中的 **taker 身份**（是自己还是对手方）
  - 分析对手方为什么参与（有没有主动意图）
  - 必须结合其他因子交叉验证
- **涉及帖子**：[2026-04-23](https://x.com/derrrrrrrq)

### 决策 3：低市值庄币筹码结构判断

- **判断规则**：庄币看筹码结构，不看地址（单看地址盲目买入风险极高）
- **条件分析**：
  - 庄控筹码充足 + 市场 FOMO（所有人都在看） + 对手盘存在 = 可能突破拉升
  - 庄无筹码（大家都冲进去买了）= 庄没动力拉，不会拉盘
  - 全市场关注 = 庄可以「突破拉升套住多头」
- **KOMA / PTB 案例**：市值不到 5M，所有人都冲进去，庄没筹码反而拉不起来
- **派发币原则**：「派发的币永远不要做多」
- **涉及帖子**：[2026-04-19](https://x.com/derrrrrrrq)、[2026-04-21](https://x.com/derrrrrrrq)

### 决策 5：现货 vs 合约使用场景划分

- **现货**：
  - 长期看好标的用现货 → 不会被洗盘出局
  - 大周期要稳赚 → 能上量的现货是最优
  - 新手建议 → 不要碰高倍合约
- **合约**：
  - 突破短线 → 合约更舒服（快进快出，杠杆放大利润）
  - 强控盘中后期 → 不参与

## 认知 / 框架更新
