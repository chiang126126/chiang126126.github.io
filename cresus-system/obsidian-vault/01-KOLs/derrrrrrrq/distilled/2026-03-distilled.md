### 决策 1：蓄水池理论完整框架（核心体系）

这是 derrrrrrrq 最核心的操盘分析框架，本月最完整阐述：

**蓄水池 = 对手盘积累量（OI 中的对手方仓位）**

```
操盘完整逻辑：

阶段一：蓄水（Cause 积累）
  主力建立多头仓位
  → 吸引做空者进入（蓄水池注水）
  → 方式：让价格缓慢涨，让空头觉得可以空；或缓慢跌制造恐慌

阶段二：挤压（Effect 释放）
  蓄水池沉淀充足后开始挤压
  → 空头止损/爆仓，提供大量买方流动性
  → 主力多头仓位获利（平多仓 = 卖出，靠空头爆仓补位）

阶段三：SC→AR（Wyckoff 经典）
  抛售高潮（SC）：主力大量卖出，制造恐慌引发踩踏
  → 自动反弹（AR）：跌太快引发自然买入
  → 期间：主力利用高流动性悄悄减少多仓

阶段四：寻新蓄水（蓄水池耗尽后）
  大行情结束后蓄水池被抽干
  → 主力需盘整蓄水，期间动作小
  → 价格下跌到特定位置吸引新空头（或用负费率吸引机器人）
```

**核心判断规则**：
- OI 充足 + 蓄水多 → 拉升 effect 大
- OI 低迷 / 蓄水不足 → 拉升 effect 小 / 假突破
- 蓄水耗尽 → 主力无法维持价格，必须等待新蓄水或弃盘

### 决策 2：主力派发操盘手法识别

**两种派发模式**：

| 模式 | 描述 | 识别信号 |
|------|------|---------|
| 拉升中出货 | 挤压空头时提供大量流动性，主力边拉边卖 | OI 大量进出，价格涨但主力仓位实际减少 |
| 假突破出货 | 在关键压力位假突破，吸引追多资金 | 突破量能大、价格慢磨关键点位后快速下跌 |

**关键识别点**：
- 流动性密集处 = 主力最容易隐藏仓位的地方
- 主力出货时机 = 流动性最高时（逼空峰值、假突破瞬间）
- 下跌往往只需几秒，上涨却需慢慢磨
- **「卖肯定在流动性最好的时候」**

## 认知 / 框架更新

- **主力核心是流动性 vs 对手盘，不是价格**：「真正深入下去你会觉得，这钱其实没那么难挣，很多人只看价格，只看高低」
- **空单 vs 多单的燃料差异**：
  - 主动做多的购买力：**永远是**多单平仓的燃料
  - 空单：只有蓄水池沉淀充足，才能成为逼空燃料；蓄水不足的空单不是燃料
- **数据源可靠性**：币安接口存在延迟/缓存，需用 AI 统计字段更新频率，以 BTC 接口验证时效性

- **大行情结束标志**：蓄水池耗尽 + 散户情绪需要修复 → 下一轮蓄水期，无大行情
