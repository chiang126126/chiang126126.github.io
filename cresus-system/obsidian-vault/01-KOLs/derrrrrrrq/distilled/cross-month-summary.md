# derrrrrrrq · 跨月总结（2026-02 ~ 04）

## 人物定位

derrrrrrrq 是**链上数据 + 主力行为分析型**短线交易者，核心特征：
- **Wyckoff 底层框架**：用「蓄水池（cause）→ 挤压（effect）」理解所有操盘行为
- **自建工具**：链上告警系统 + AI 辅助策略 + 数据管道（非依赖三方平台）
- **极短线执行**：只吃趋势，盘整不持仓，单笔 1 小时内完成
- **不赌方向**：只在有主力信号确认的情况下进场，无信号绝不开单
- **技术迭代者**：将交易系统类比软件工程（poll → epoll 优化），持续复利迭代

## 核心交易框架

### 框架一：蓄水池理论（最核心体系）

```
核心公式：流动性 × 对手盘 = 操盘者的利润空间

蓄水池 = 对手盘积累量（合约中的反向仓位）

阶段判断：
┌──────────┬────────────────────────────┬──────────────────┐
│ 阶段     │ 信号                       │ 操作             │
├──────────┼────────────────────────────┼──────────────────┤
│ 蓄水期   │ OI 缓慢积累，价格震荡       │ 等待，不进场     │
│ 挤压期   │ OI 急剧减少 + 价格爆发      │ 顺势短线进场     │
│ SC→AR   │ 大量现货砸盘后快速反弹      │ 等 40 分钟后进   │
│ 耗尽期   │ OI 低迷，价格脆弱，负费率   │ 绝对不做多       │
│ 新蓄水期 │ 盘整吸引新空头/机器人       │ 观望等新信号     │
└──────────┴────────────────────────────┴──────────────────┘
```

**蓄水池充足/不足的判断**：
- 充足：OI 有显著积累，前次行情蓄水多，effect 会大
- 不足：OI 与最低点持平，拉升 effect 小，假突破高风险

**弃盘识别**：
- 价格无法吸引流动性 → 主力选择盘整很久再试探 → 或直接弃盘

### 框架二：主力行为四步识别法

```
步骤 1：确认主力仓位方向
  - 链上地址：大额转入做市地址 = 准备拉升
  - 大额现货转到 CEX 热钱包 = 准备派发/砸盘
  - 负费率 + 机器人持续进场 = 主力用机器人接盘出货

步骤 2：判断操盘阶段
  - OI 积累 → 蓄水期
  - OI 暴降 + 价格爆拉 → 挤压期（effect 释放）
  - OI 低迷 + 价格震荡 → 耗尽/弃盘期

步骤 3：识别派发手法
  - 拉升中出货：高流动性时主力边拉边卖
  - 假突破出货：在关键压力位做假突破，引爆追多后快速平仓

步骤 4：等待信号触发
  - 突破前高 + OI 新低 = 高 ROI 做多信号
  - 主力打压告警 + 盘面激活 = 顺势做多
  - 趋势反转信号 = 立即离场
```

### 框架三：短线执行纪律体系

```
进场条件（需满足至少 2 项）:
1. 突破前高 + OI 新低（高 ROI 信号）
2. 主力打压告警 + 量能放大（盘面激活）
3. 链上大额转入做市地址（外部催化确认）
4. 外部攻击事件后 40 分钟（恐慌后的安全反弹）

不进场条件（任意 1 项 = 放弃）:
1. 强控盘中后期（全网关注、所有指标可能失效）
2. 蓄水池耗尽（OI 极低迷）
3. 派发阶段（负费率 + 机器人接盘 + 项目方分批出货）
4. 信号不在系统告警范围内（临时起意开单）

执行规则:
- 只吃趋势，不做盘整
- 趋势反转信号出现 → 立即平仓，不恋战
- 止损：前低
- 止盈：流动性激活后快速走（不吃尾巴）
- 目标持仓时间：1 小时内
```

## 可用于 DeepSeek 系统提示的判断规则

### 蓄水池状态判断

```
蓄水充足（可进场做多信号）:
1. OI 有显著积累（较前期低点增加 > 30%）
2. 负资金费率持续（空头被吸引进场）
3. 价格震荡但未大幅下跌（主力维持蓄水成本）

蓄水耗尽（禁止做多）:
1. OI 降至历史低区（1 个月内低点）
2. 价格波动性显著降低
3. 负费率 + 机器人接盘（主力用机器人出货）

挤压开始（做多最佳时机）:
1. OI 突然快速下降（空头开始爆仓）
2. 价格快速上涨（squeeze 启动）
3. 突破前高 + OI 新低 = 高 ROI 进场
```

### 主力派发识别

```
派发信号（满足 2 项以上 → 不做多）:
1. 大额现货转移到 CEX 热钱包
2. 负资金费率持续（机器人接盘信号）
3. 链上项目方地址分批减少
4. 价格假突破压力位后快速回落
5. 强控盘 + 全网注意力集中（指标全部失效风险）
```
