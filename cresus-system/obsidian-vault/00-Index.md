## 📋 工作流

### 每天做一次
1. 刷 Twitter / Discord 看到好帖 → 丢进 `#distillation` 频道
2. 让 Claude Code 按 [[05-Templates/distillation-template]] 蒸馏
3. Claude 自动把笔记存入对应 KOL 目录

### 每周做一次
1. 回顾本周蒸馏的新笔记
2. 更新对应 KOL 的 `framework.md`
3. 如果发现新的市场概念，补进 `04-Market-Concepts/`

### 每月做一次
1. 全量复盘 6 位 KOL 的 framework，检查有没有跟市场脱节
2. 更新 `02-Frameworks/cresus-master-framework.md`
3. 重新部署 bot（system prompt 变了）
